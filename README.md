## A) User Registration and Login

The website provides users with a registration form and a login form upon entering the website. Users are also given a login option to prevent the need to create multiple accounts.

![Login Screenshot](Project-Media/Login.png)

**Purpose:** Allows new users to create accounts and log in to the website.  
**Functionality:** Users fill out a registration form with their email, password, and other required information. The system validates the data.  
**Dependencies:** Relies on a backend user management system and possibly a third-party email service for verification.

## Design and Layout

![Page Design](Project-Media/Page-Design.png)

The website follows a clean and minimalistic layout. The content is well-organized, with clear sections for different topics.  
The navigation menu is typically located at the top, allowing users to access different pages easily.  
The homepage features a hero section with a prominent call-to-action (CTA) button, inviting visitors to explore further.  
The use of white space ensures readability and prevents clutter.
